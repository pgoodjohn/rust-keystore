## Installing :clapper:
### Prerequisites
To build keystore you will need [Rust](https://www.rust-lang.org/), and [cargo](https://crates.io/)

### Setup

Clone the project:

```
git clone git@github.com:pgoodjohn/rust-keystore.git
cd rust-keystore
```

Run the application:

```
make start
```
